# 🚀 START HERE - WA Automation Platform

## ⚡ Quick Start (Choose One)

### Option 1: Interactive Setup Wizard (Recommended)
```bash
setup-wizard.bat
```
This launches an interactive menu to guide you through setup.

### Option 2: Manual Quick Start
```bash
# 1. Install dependencies
install-all.bat

# 2. Verify setup
verify-setup.bat

# 3. Start development
start-dev.bat
```

---

## 📋 What You Need

Before starting, ensure you have:

1. ✅ **Node.js 18+** installed
2. ✅ **Firebase CLI** installed (`npm install -g firebase-tools`)
3. ✅ **Service Account Key** from Firebase Console
   - Download from: Firebase Console → Project Settings → Service Accounts
   - Save as: `functions/serviceAccountKey.json`
